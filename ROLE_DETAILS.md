* Portfolio / LinkedIn (optional)

  * Co-founder  
  * Learning  
  * Networking

## **My suggestion (important)**

Don’t show too many fields on one screen. Use **2-step signup**:

### **Step 1**

Basic signup:

* Name  
* Email  
* Mobile  
* Password / Create Password

### **Step 2**

Role-based details

This will improve signup completion rate.

Also you can add **Skip for now** for optional profile completion.
